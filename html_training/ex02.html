Basic Html tags

Every HTML document has following things:

Headings: Used to provide headings in doc with tags from < h1>(biggest) to < h6>(smallest)

Sections:Blocks that replaced divs within html doc.

Paragraphs:similar to divs but used to give small amount of data.provided with p

Spans:Inline content are Spans.Used with tag called < Span>
